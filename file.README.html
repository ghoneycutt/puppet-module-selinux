Module description

This module manages SELinux and by default will enable it.

Setup

What selinux affects

Manages SELinux and its configuration. It manages the file /etc/selinux/config and optionally manages the policycoreutils-python package.

Setup requirements

This module requires stdlib.

Beginning with selinux

Examples

Enable SELinux
include ::selinux
Disable SELinux
class { '::selinux':
  mode => 'disabled',
}

Usage

Minimal and normal usage.

include ::selinux

Limitations

This module has been tested to work on the following systems with Puppet versions 5 and 6 with the Ruby version associated with those releases. Please see .travis.yml for a full matrix of supported versions. This module aims to support the current and previous major Puppet versions.

  • EL 5
  • EL 6
  • EL 7
